这是一个简易的网上书店项目  
它的主要目的不是为了完成具体的功能, 而是体验将简单的JavaWeb项目转换为使用SSM框架的过程。   
所以首先它将通过servlet+jsp+jdbc来实现
然后依次加入或更换为SpringMVC、Mybatis、Spring。
